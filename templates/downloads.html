Conda packages

Mantid is available in a number of conda packages via anaconda.org on the mantid channel.

The following packages are available:

  • mantid: This package includes the core capabilities, including algorithms and workspaces, availiable via python
  • mantidqt: Currently a work in progress, nightly only package, this includes the core Python library and scientific interfaces without the Workbench interface.
  • mantidworkbench: Currently a work in progress, nightly only package, this is the full Mantid suite, including the workbench GUI. This should be used if you're unsure what Mantid is, or which package you need.
